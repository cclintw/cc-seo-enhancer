== Description ==

CC SEO Enhancer provides focused SEO helpers for WordPress sites, including meta descriptions, Open Graph metadata, schema.org JSON-LD output, optional tracking snippets, author social profile fields, and custom robots.txt Disallow rules.

The plugin uses the WordPress core sitemap at wp-sitemap.xml instead of generating a separate sitemap file.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the Plugins screen in WordPress.
3. Configure settings under Settings > CC SEO Enhancer.

== Frequently Asked Questions ==

= Does this plugin generate sitemap.xml? =

No. It uses the WordPress core sitemap at `/wp-sitemap.xml`.

= Does this plugin create a physical robots.txt file? =

No. It customizes WordPress' virtual robots.txt output.
